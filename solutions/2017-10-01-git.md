- Visual [Git Cheatsheet](https://raw.githubusercontent.com/nerdgirl/git-cheatsheet-visual/master/gitcheatsheet.png)
- Reference [Git Documentation](http://git-scm.com/docs)
- Create repository in folder `git init`
- Clone existing repository  `git clone LINK`
- Add any new files `git add .`
- Commit changes `git commit -m 'Message'`
- Push changes `git push`
- See log `git log`
- See changes `git status` or `git diff`
